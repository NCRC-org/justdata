# LoanTrends

Renders trend charts from CFPB's HMDA Quarterly Data Graph API:
applications, originations, denials, credit scores, and related series.

## Blueprint

- URL prefix: `/loantrends`
- File: `blueprint.py` (`loantrends_bp`)
- Routes: `/` (dashboard), `/api/dashboard-data`, `/analyze`,
  `/progress/<job_id>`, `/results/<job_id>`, `/report/<job_id>`,
  `/api/available-graphs`.

## Data sources

**No BigQuery.** All data comes from CFPB's HMDA Quarterly Data Graph
API (`config.QUARTERLY_API_BASE_URL`). Responses are cached in-process
for 24 hours (`data_utils._cache`).

## Reports

`report_builder.py` (single module) assembles the report. Charts are
generated by `chart_builder.py`. `build_static_site.py` and
`generate_static_html.py` produce a static HTML version under
`static_site/`.

## Templates

`templates/`:
- `dashboard.html`, `loantrends_analysis.html`, `report_template.html`,
  `analysis_template.html`
- `partials/` — `_loantrends_report_head.html`,
  `_loantrends_report_main.html`, `_loantrends_report_scripts.html`
