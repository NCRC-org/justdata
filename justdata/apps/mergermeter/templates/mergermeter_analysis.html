Bank A (Acquirer) Assessment Areas

Generate assessment areas automatically from branch locations

This will query all branch locations for Bank A and create assessment areas grouped by CBSA/metro area. Only counties where the bank has branches are included.

Note: Assessment areas include only counties where the bank has physical branch presence.

Requires RSSD number. Assessment areas will be grouped by metro area (CBSA).
