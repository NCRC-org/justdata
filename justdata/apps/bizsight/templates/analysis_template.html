Important: Connecticut Data Change

What Changed?

Starting in 2024, Connecticut changed how it reports business loan data. Instead of using counties (like Fairfield County, Hartford County, etc.), the state now uses planning regions (like "Greater Bridgeport Planning Region" or "Capitol Planning Region").

Why this matters: Because the data structure changed, we cannot combine 2020-2023 data (which uses counties) with 2024 data (which uses planning regions) in a single report. They're like comparing apples and oranges.
