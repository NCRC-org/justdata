# BranchMapper

Interactive map of bank branch networks with census-tract overlays and
multi-sheet Excel export. Lets staff draw polygons or pick counties/MSAs
and inspect FDIC SOD data overlaid on demographic geography.

## Blueprint

- URL prefix: `/branchmapper`
- File: `blueprint.py` (`branchmapper_bp`)
- Routes cover the map page, county/state lookup APIs, and bounds-based
  branch queries.

## Data sources

- BigQuery: `justdata-ncrc.branchsight.sod` (FDIC Summary of Deposits — same
  dataset as BranchSight; `config.DATASET_ID = "branchsight"`)
- Census tract overlays from a Mapbox tileset (`editengine.census-tracts`,
  ~60k features) — generated by `scripts/generate_census_tileset.py`.
- Reuses `justdata.apps.branchsight.data_utils.expand_state_to_counties`.

## Reports

No web/PDF report. Export is a multi-sheet `.xlsx` (Notes, Bank Branches,
Changes) generated client-side via SheetJS, respecting the drawn polygon
filter.

## Templates

`templates/`:
- `branch_mapper_template.html` (main page)
- `partials/` — `_branchmapper_head.html`, `_branchmapper_scripts.html`,
  `_branchmapper_scripts_footer.html`, `_filters.html`, `_map_container.html`

## Notes

- Census tract utilities in `census_tract_utils.py`.
- FDIC OSCR event parsing in `data_utils.parse_fdic_events`.
- `pdf_methods.py` exists but the Excel path is the supported export.
